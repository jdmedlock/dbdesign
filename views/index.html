DBDesign - Mongo & Mongoose Demonstration

Overview

DBDesign demonstrates the basic functionality provided by the MongoDB database and then duplicates it to allow the native Mongo code to be compared and contrasted to comparable Mongoose code.

This is implemented as a series of Express routes which implements an API supporting the setup and reinialization of the Mongo database and test data.

API

  • Setup - http://localhost:3000/setup/populatedb

    If the Accounts database delete and reload all test data. If the database doesn't exist then define it and load test data.

  • Mongo Tests
    1. findall - http://localhost:3000/mongo/findall

      Retrieve and list all records using native Mongo calls.

  • Mongoose Tests
    1. findall - http://localhost:3000/mongoose/findall

      Retrieve and list all records using Mongoose.
